## 🎬 영상 파이프라인 (확립됨)

```
가사 → Suno 곡 → 루마 루프영상 → overlay_nowplaying.py → 완성
```

- **루프 영상은 Luma로** (Grok·Kling 실패 확인). 반드시 **Loop 기능 ON**
- 앵커 이미지가 **시선 방향을 결정** (프롬프트로는 안 바뀜)
- "자는" 컨셉은 고개 흔들림 유발 → **깨어있는** 컨셉으로
- 오버레이: `scripts/overlay_nowplaying.py` (제목 + 진행바 + 시간)
