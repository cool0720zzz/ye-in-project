# A01 「네온 (Neon)」 — 가사 + Suno 세팅

- **세트**: A (도시의 밤 풍경)
- **주제**: 이별 후 **일정하지 않은 마음.** 괜찮다가 아니다가, 생각조차 안 나는 밤이 있다가 갑자기 훅 온다 — **그 예측 불가능성**이 곡의 정체
- **핵심 은유**: **네온은 켜져 있는 게 아니라 깜빡인다.** 마음의 점멸 = 네온의 점멸

### ⭐ 극성 뒤집기 대구 (곡의 중심 장치)
같은 문장을 **한 단어(켜진↔꺼진)만 뒤집어** 두 지점에서 반복한다.

| 지점 | 대사 |
|---|---|
| **V2** | 어느 한쪽이 꺼져야 켜진다면 / **불이 `켜진` 쪽은 나여야 하는 건가** |
| **Bridge** | 이제 그 이름에 불이 켜졌으니 / **불이 `꺼진` 쪽이 나여야 하는 건가** |
| **Outro** | 켜지고 꺼지고 / 함께 꺼지고 켜지는 → 단정 대신 **점멸 상태로 유보** |

> 💡 **같은 사람이 두 번 다르게 묻는다** = 마음이 흔들린다는 게 설명 없이 구조로 드러남.
> 진폭을 *설명*하지 않고 *실연*하며, 후렴 밖에 두 번째 훅까지 만듦.
> 💡 `나였던 건가`(회고) → `나여야 하는 건가`(**당위에 대한 의문**)로 바꿔 아픔이 커짐.
> ⚠️ 주제는 **점멸(오락가락)**. 한 상태로 고정하는 개작(계속 그립다 / 계속 억울하다)은 주제를 흐림.
- **BPM/무드**: 95, **마이너 키 멜랑콜리 시티팝**, 쓸쓸하고 나른한 야간
- **사운드 레퍼런스**: 마츠바라 미키 「Stay With Me」(真夜中のドア) — 시티팝 편성 그대로지만 **마이너 키**라 쓸쓸함
- **언어**: 한국어 (후렴 포인트에 "Neon" 영어 스파이스 1회)

## 개작 노트 (v4 최종, 2026-07-20)

초고가 **장면을 설명만 하고 변형하지 않아** 상투적이었음 → 4차 개작으로 확정.

### 상투구 제거
| 폐기한 상투구 | 대체 | 기법 |
|---|---|---|
| 네온 불빛 아래 나 홀로 걷는 밤 | **네온빛이 얼굴에 잠깐 머물다 가** | 빛이 잠시 머물다 떠나는 것으로 |
| 분홍빛 파란빛 물든 거리 | **젖은 길 위로 색이 번지고 / 쏟은 사람은 여기 없는데** | 색번짐 → 부재의 증거 |
| 네 목소리가 들려와 | **간판의 글자가 하나씩 뭉개져 / 시야가 그 이름을 지우는 밤** | 간판 소등 → 자기 소멸 |
| 반짝이다 또 흐려져 | **도시는 밤마다 다시 켜지는데 / 나는 어디쯤부터 꺼져있었나** | 도시↔나 반전 (곡의 축) |
| 이 밤을 오래 기억할게 | **이제 그 이름에 불이 켜졌으니 / 불이 꺼진 쪽이 나여야 하는 건가** | 교환 논리를 사람에게 적용 |
| 하지만 어쩔 수 없지 | (삭제) | 체념으로 감정을 닫아 여백 소실 |

### 협업 중 채택된 개선
| 개선 | 효과 |
|---|---|
| `누가 내 이름을` → **`시야가 그 이름을`** | 지우는 주체가 외부 → **내 안**. 슬픔이 안에서 일어남 |
| `나는 자꾸 없는 쪽을 봐` → **`시선은 자꾸`** | 내가 보는 게 아니라 **시선이 제멋대로** = 통제 불능 |
| `어디쯤에서` → **`어디쯤부터`** | 시점이 아니라 **지속**을 가리켜 여백 확대 |
| Verse 2 전면 교체 → **꺼짐↔켜짐 교환 관계** | 빛의 총량이 정해져 한쪽이 꺼져야 켜짐. 벌스가 후렴 질문을 세팅 |
| Verse 2 전면 교체 → **꺼짐↔켜짐 교환 관계** | 빛의 총량이 정해져 한쪽이 꺼져야 켜짐 |
| ⭐ **V2·Bridge 마지막 행을 대구로 통일** | `불이 **켜진** 쪽은 나여야 하는 건가` ↔ `불이 **꺼진** 쪽이 나여야 하는 건가` — **한 단어만 뒤집어** 같은 질문을 두 번 |
| `나였던 건가` → **`나여야 하는 건가`** | 회고 → **당위에 대한 의문**. "내가 그랬었나"보다 "왜 내가 그래야 하지"가 더 아픔 |
| `꺼져버린` → **`꺼져있었나`** | 사건(완료) → **지속된 상태** + 건조한 어미로 절제된 톤 확보 |
| 아웃트로 마감 | **`빛나던 이름을 찾고 있어`** 로 **`이름` 실 매듭** (V1 지움→Bridge 보임→Outro 찾음). 이름을 빛으로 처리(`빛나던`)해 이미지 유지 — 곡의 유일한 직접성이나 설명조로 안 떨어짐 |
| ⭐ 프리코러스 신호등 통일 (`초록불은 오지 않고` / `빨간불이 바뀌지 않고`) | **못 잡은 사인 + 못 벗어남**을 신호등 이미지로. 네온"사인"의 '사인' 이중의미를 *설명 없이* 점멸 축 안에 녹임 (초록불=못 감 / 빨간불=여전히 못 감 → 이별 후 못 떠남) |
| `봐` → `향하고` (프리코러스 종결) | 두 프리코러스가 `-고/-고`로 끝나 후렴에 매달려 흘러듦 = 미완의 지속감 |
| ⭐ `이름` 3박자 실 | V1 `그 이름을 **지우는**` → Bridge `익숙한 이름이 **보일**` → Outro `빛나던 이름을 **찾고**` = 지움→보임→찾음. 아웃트로가 이 실을 거둠 |

> ⭐ **질문 어미 통일**: `꺼져있었나` / `깜빡거리던 건가` / `나여야 하는 건가`
> 물음들이 같은 결로 묶이면서 곡 전체가 **하나의 질문**으로 수렴함.
> 특히 후렴 끝행 **`깜빡거리던 건가`**는 곡의 핵심 은유(네온=점멸)를 후렴에서 **직접 호명** — "나도 깜빡이고 있었나". **세 후렴 모두 이 물음으로 닫혀** 점멸 축이 곡을 관통함(집요한 아픔). 아웃트로는 후렴의 물음을 반복하지 않고 **`이름` 실을 거두며 마감**(`빛나던 이름을 찾고 있어`) — 점멸(후렴)과 이름 추적(아웃트로)이 각자 닫힘.

### ⭐ "너" 심기 — 이별 서사 가독성 (2026-07-22)

**문제**: 외부 독자가 이 곡을 **이별이 아니라 "도시 소진"으로 오독**했다.
`그 이름`·`익숙한 이름`만으로는 상대의 존재가 안 읽힘 — **과잉 절제**였다.

**처방 — 최소 개입**: 프리코러스 두 줄에만 심는다.

| 지점 | 이전 | 현재 |
|---|---|---|
| PC1 | 시선은 자꾸 `없는 쪽`을 향하고 | 시선은 자꾸 **`너와 걷던 쪽`**을 향하고 |
| PC2 | 시선은 자꾸 `없는 쪽`을 향하고 | 시선은 자꾸 **`네가 있던 쪽`**을 향하고 |

> ⚠️ **과교정 폐기 기록**: 한때 V1·후렴·브릿지·아웃트로까지 6~7군데를 고쳐 "너"를 넣었으나 전부 되돌림.
> 그 과정에서 **후렴 훅**(`깜빡거리던 건가` 7음절 → 22음절 2행), **극성 뒤집기 대구**(`불이 켜진` → `지금 켜진`으로 짝 깨짐), **`이름` 실**(아웃트로)이 전부 무너졌다.
> **원칙: 가독성 문제는 가장 적은 지점에서 푼다. 훅과 중심 장치는 건드리지 않는다.**
> 💡 후렴을 길게 늘리는 건 특히 위험 — Suno가 긴 코러스엔 매번 다른 멜로디를 붙여 훅이 안 생긴다.

### 곡의 두 축

**① 이미지 축 — 흐림 → 번짐, 그리고 `이름` 실**
시야가 뭉개짐(흐림) → 색이 번짐 → 아웃트로에서 **다시 흐린 시야 속** 이름을 찾음. 눈물(흐림)은 끝까지 안 걷히고, 그 속에서 `이름`을 좇으며 닫음

**② 구조 축 — 점멸: 같은 질문의 극성이 뒤집힌다**
| 지점 | 대사 | 상태 |
|---|---|---|
| Chorus ×3 | `**나는**→**너는**→**내가** 어디쯤부터 꺼져있었나` | **주어 오락가락** — 내 탓/네 탓 사이를 흔들리다 강조형 `내가`로 착지. 점멸 축을 주어에까지 확장 (`너는`이 브릿지의 `익숙한 이름`을 미리 깖) |
| V2 | `불이 **켜진** 쪽은 나여야 하는 건가` | 내가 아직 켜진 쪽인가 |
| Bridge | `불이 **꺼진** 쪽이 나여야 하는 건가` | **극성 반전** — 이번엔 꺼진 쪽인가 |
| Outro | `켜지고 꺼지고 / 함께 꺼지고 켜지는` | 단정하지 않고 **점멸 상태로 유보** |

> 💡 같은 사람이 두 번 다르게 묻는 것 자체가 **마음이 흔들린다는 증거**.
> 진폭을 설명하지 않고 **구조로 실연**한다.

> ⚠️ **트레이드오프 기록 (복구 카드 2장)**
> ① 브릿지 교체로 유일한 감정 전환(`지워진 게 아니라 / 흘러간 거라고 부를래`, 재명명 기법)이 빠짐 → 곡이 과하게 어두우면 되살릴 것
> ② 후렴 2행: `빌려온 빛이라` → `지나가는 빛들은` → **`스치는 빛들은`**(현재). `스치는`이 앞줄 `머물다 가`의 촉감과 붙어 개선. 그래도 밋밋하면 이유를 주는 `빌려온 빛이라`로 복구 가능
