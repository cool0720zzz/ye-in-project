- **세트**: A (도시의 밤 · 1인칭)
- **감정**: **소진·공허** — 사랑 정서 없음. 열심히 하는데 어디로 가는지 모르겠는 상태

## 설계 노트

### ⭐ 구조 설계 — 형식이 주제를 따른다
```
A01 네온     Intro - V1 - Pre - Chorus - V2 - Pre - Chorus - Bridge - Chorus - Outro
A02 워커홀릭  Intro - V1 - Chorus - V2 - Chorus - Bridge - [Synth Solo] - Chorus - Outro
```
- **프리코러스 제거**: 주제가 "매일 똑같다"인데 구조에 장식이 많으면 안 어울림.
  단조로운 골격일수록 **유일한 변화인 브릿지(맥주)가 도드라짐**
- **벌스를 6행으로 확장**: 프리코러스에 있던 커피·미루기를 흡수. 벌스가 길어지며 **버티는 시간의 지겨움**이 형식으로 드러남
- **[Synth Solo] 삽입**: A01엔 간주가 없어 구조로 차별화. **색소폰이 아니라 신스인 이유** — sax는 본질적으로 따뜻하고 인간적인 악기라 이 곡의 차가운 방향과 정면 충돌. **차갑고 유리 같은 FM 신스 리드(DX7 계열)**로 대체해 형광등 아래 공허를 길게 늘림
  > ⭐ 결과적으로 **A세트에서 유일하게 sax가 없는 곡**이 됨 (네온=후렴 sax / 회식=소프트 sax / 번화가=sax stabs). 편성 자체가 차별화 장치

> ⚠️ **13곡이 같은 골격이면 안 된다.** 롱폼 연속 재생 시 단조로움이 바로 드러남.
> 구조는 미리 정해두지 말고 **곡의 정서 곡선에 맞춰 매번 설계할 것.**

### ⭐ 사운드 방향 — 차갑게 (네온과 차별화)
네온과 워커홀릭이 둘 다 "warm breathy city pop"이라 무드가 겹치는 문제 → 워커홀릭을 **차갑게** 재정의.
- **네온** = 따뜻·나른·lush, wistful (웜 발라드)
- **워커홀릭** = **cold minimal, synth-driven, mechanical repetitive, deadpan, sparse sterile** — 주제(매일 똑같음)를 **기계적 반복 그루브**로 구현
- 정서 끝점도 다름: 네온은 *이름을 찾음*(그리움), 워커홀릭은 *똑같은 내일에 갇힘*(탈출 없음)

### 중심 장치 — 탈출 욕구조차 없는 무감각
"때려치고 싶다"는 **아직 감정이 남아 있는 상태**다. 이 곡은 그 단계마저 지난
**아무 느낌이 없는** 자리를 그린다 — 감정어를 쓰지 않고 **끝없이 반복되는 사실**(`어제 한 일이 오늘 다시 와 있고` / `똑같은 하루일 것 같아`)만 나열해 소진을 실연한다. 소진의 가장 깊은 지점.

### ⭐ 마지막 후렴 — 같은 구조, 장소만 이동 (사무실 → 집)
| | 후렴 1·2 | **후렴 3 (마지막)** |
|---|---|---|
| 바람 | 끝내면 **집에 가려고** 했는데 | 끝내면 **침대에 눕고** 싶었는데 |
| 안 끝나는 것 | 끝나는 **칸**이 없나 봐 (업무) | 끝나는 **집안일**이 없나 봐 (가사노동) |

나머지 3행은 **그대로 두고 장소만 이동**시켜, **퇴근해도 끝이 없다**는 걸 설명 없이 보여준다.
「네온」의 극성 뒤집기 대구(`켜진`↔`꺼진`)와 같은 계열 — **최소 변경 반복으로 의미를 조이는** 기법.
> 💡 도망칠 곳이 없다는 게 **형식 자체로** 드러남. 마지막 후렴이 단순 반복이 아니라 **한 번 더 조여드는** 자리가 됨

### ⭐ 볼빨간사춘기 「워커홀릭」과의 차별화
| | 볼빨간사춘기 | **이 곡** |
|---|---|---|
| 시점 | 2인칭 메시지 (위로·응원) | **1인칭 관찰** |
| 정서 | **일탈** — "때려쳐야겠다", 당당하게 살자 | **소진** — 탈출 욕구조차 없음 |
| 에너지 | 밖으로 튀어나감 | 안으로 잠김 |
| 결말 | 현실에 좌절 말고 당당하게 | **맥주 한 캔으로 버팀** |

> ⚠️ **금지**: 일탈·때려침 서사, 응원형 메시지, "당당하게 살자"류 결론 — 전부 볼사 영역

### 진폭 — 맥주 한 캔
```
오늘 내 손으로 고른 건
결국 캔맥주 하나뿐
```
하루 종일 남이 시킨 것만 하다가, **맥주 한 캔이 유일한 자기 선택**.
거창한 위로가 아니라 **아주 작은 것 하나로 버티는** 현실 — 이게 곡의 온도를 만든다.
`결국`이 체념을 싣는다.

### 아웃트로 — 한 캔 더 + 똑같은 내일 + 밤의 끝을 잡음
```
다시 맥주 한 캔 더 따고
잠들고 나면 내일도
똑같은 하루일 것 같아
이 밤의 끝을 잡아
```
- `다시 맥주 한 캔 더 따고` — **행동만 놓는다.** 브릿지에서 고른 그 한 캔이 두 캔째가 됨. 왜 더 마시는지 **설명하지 않음**
- `잠들고 나면 내일도 / 똑같은 하루일 것 같아` — 후렴의 "어제→오늘→내일" 순환이 **하루 전체의 순환**으로 확장돼 북엔드
- ⭐ `이 밤의 끝을 잡아` — 곡에서 **유일하게 화자가 무언가를 붙잡는** 행동. 잠들면 똑같은 내일이니 **잠들기 전 이 밤의 끝자락만이 내 것**
- ⭐ **한 캔 더 따는 행위 = 밤을 붙잡는 방법.** 두 줄이 인과로 이어지되 그 인과를 **말하지 않는다**
  > 폐기: `이 한 모금 마시려고 / 내가 일하는 거였지` — 맥주의 **의미를 설명**해버려 삭제. 행동만 남기니 더 강해짐
- ⚠️ 초기안은 질문(`눈꺼풀이 무거웠나 / 맞춰뒀던가`)으로 닫았음. 현재는 행동+순환 (네온=질문/그리움 ↔ 워커홀릭=순환/갇힘으로 차별화)

> ⚠️ **폐기 기록**: `이 한 잔이 오늘의 낙이지`(의미를 설명), `이미 체력이 방전이라`(이유를 설명)는
> 1.5절 목소리 원칙 위반으로 삭제. 설명하는 순간 다음 줄의 힘이 빠진다.

### 가이드 준수 확인
| 원칙 | 적용 |
|---|---|
| 감정 앵커 최소 1개 | `오늘은 답이 없어` / `똑같은 하루가 오겠지` — 정서가 직접 잡히는 지점 |
| 논리 명제 금지 | 규칙·추론 없음. 전부 관찰과 실제로 드는 생각 |
| **감정은 한 상태로 고정 금지** | Bridge의 맥주 한 캔이 반대편(작은 온기)을 담당 — 공허로만 끝나지 않음 |
| 사물이 감정을 지게 | 꺼지는 층 불, 줄어드는 발소리, 식은 커피, 어제 입던 옷, **캔 따는 소리** |
| 상투구 | 블랙리스트 0건 |

### 시간 감각 상실 (V2)
`저녁은 언제 먹었는지 / 창밖이 언제 어두워졌는지 / 의자에서 일어난 건 언제였지`
— **"언제" 세 개 병렬.** 끼니·해질녘·기립 시점을 다 놓친 상태. 시간이 뭉개진 감각으로 소진을 실연한다. `언제`를 반복해 형식으로 못 박음.
