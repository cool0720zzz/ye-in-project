## 설계 노트

### ⭐ 구조 — 형식이 소음의 소멸을 따라간다
```
A01 네온     Intro - V1 - Pre - Chorus - V2 - Pre - Chorus - Bridge - Chorus - Outro
A02 워커홀릭  Intro - V1 - Chorus - V2 - Chorus - Bridge - [Synth Solo] - Chorus - Outro
A03 회식     Intro - V1 - Chorus - V2 - Chorus - Bridge - Outro   ← 마지막 후렴 없음
```
- **후렴이 끝에 안 돌아온다.** 시끌벅적하던 게 그냥 사라지듯 끝남
- **섹션이 점점 짧아진다**: 벌스 4행 → 브릿지 4행(짧은 행) → 아웃트로 3행
- 브릿지에서 **소리가 물리적으로 줄어드는** 지점(화장실 문)이 곡의 전환

### 중심 장치 — 외워서 웃는다
```
웃는 타이밍은 이제 알겠어
맞장구는 몸이 먼저 쳐
```
감정이 아니라 **학습된 반응**. "즐겁지 않다"고 말하지 않고,
**웃음이 자동화됐다는 사실**만 놓는다.

> ⭐ **`-고` → 종결형 교체 (2026-07-22)**: 원안 `알겠고 / 치고`는 후렴 한가운데 **닫힌 모음(오)이 연속 2회**라 멜로디가 눌렸다.
> `알겠어 / 쳐`로 바꿔 **열린 모음(어)**으로 착지 + 부드러운 어/어 각운까지 생김.
> 의미도 이득 — 연결형(`~고`)이 종결형이 되며 **두 개의 납작한 사실**로 떨어져 자동화된 톤에 더 맞는다.
> ⚠️ A세트 후렴 중 `-고` 연속은 이 곡뿐이었다. 다른 곡 후렴은 착지 행이 이미 열려 있어 손대지 않음.

`누가 내 이름을 부르면 / 그제야 고개를 들어` — 자기 존재를 잊고 있었다는 걸 행동으로만 보여줌.

### 몸의 감각 — 냄새 (2026-07-22)
```
머리엔 고기 냄새가 배어
```
기존 몸 묘사가 전부 **동작**(웃기·맞장구·고개 들기)뿐이라 **감각이 없다**는 지적에 대응.
- 회식의 감각은 **냄새**다 — 가장 한국적이고 가장 오래 남는 것
- ⭐ **내가 아무것도 안 해도 배어든다.** 술이 알아서 채워지듯 냄새도 알아서 밴다 → **수동성이 한 겹 더해짐**
- 집에 가도 안 지워지므로 **회식이 몸에 묻어 따라온다** (「워커홀릭」이 일을 집까지 끌고 가는 것과 같은 계열)

### ⭐ 브릿지 — 거울 속에 아빠가 있다 (2026-07-22)
```
거울 앞에 텅 빈 채 웃는
어릴 적 아빠의 표정이 있네
```
`직장생활하는 어른이 있네` → **교체.**
- **설명 → 이미지**: 전자는 "내가 어른이 됐다"를 *말한다*. 후자는 거울에 비친 얼굴을 *보여준다*
- ⭐ **A세트 전체에서 유일하게 과거(시간)가 들어오는 지점.** 나머지 전곡이 현재 습관형이라, 여기서만 시간축이 생기며 곡이 깊어진다
- **세대 대물림**이 공짜로 붙는다 — 내가 참는 얼굴이 아빠가 참던 얼굴과 같다
- 거울 장치가 완성됨: 거울을 보는데 **나 대신 아빠가 보인다**

> 💡 `어릴 적 보았던 아빠의 표정` → `어릴 적 아빠의 표정`으로 트림. `보았던`은 `어릴 적`에 이미 들어있는 중복 정보

### 주체 이동 (가이드 ②-b)
```
표정이 먼저 풀린다
```
내가 표정을 푸는 게 아니라 **표정이 먼저** 풀린다.
참고 있던 게 몸에서 먼저 무너지는 것 — 통제 밖에서 일어나는 일로 처리.

### 아웃트로 — 가면을 다시 건다
```
손을 씻고
표정을 다시 걸고
문을 연다
```
`표정을 걸다` = 옷걸이에 걸듯 얼굴을 착용. 문을 열면 다시 소음.
**해결 없이 반복으로 닫는다.**
