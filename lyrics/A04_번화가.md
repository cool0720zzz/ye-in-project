## 설계 노트

### ⭐ 구조 — 후렴 선행
```
A01 네온     Intro - V1 - Pre - Chorus - V2 - Pre - Chorus - Bridge - Chorus - Outro
A02 워커홀릭  Intro - V1 - Chorus - V2 - Chorus - Bridge - [Synth Solo] - Chorus - Outro
A03 회식     Intro - V1 - Chorus - V2 - Chorus - Bridge - Outro   (끝 후렴 없음)
A04 번화가   Intro - Chorus - V1 - Chorus - V2 - Bridge - Chorus - Outro   ← 후렴부터 시작
```
번화가는 **들어서는 순간 압도당하는** 곳. 벌스로 천천히 쌓지 않고 **바로 후렴이 터지는** 게 장면에 맞다.

### ⭐ 4개 층 (읽어야 보이는 설계)

**① 화자는 고개를 들지 못한다 — 시선의 높이**
```
줄 선 사람들의 신발이 모두 다른데
```
왜 하필 **신발**인가. 얼굴을 안 보고 있기 때문이다. 시선이 발치에 있다.
`가방끈만 꼭 쥐고`, `프레임 밖으로`도 같은 높이 — **사람과 눈을 마주치지 않는다.**

**② 모든 접촉이 '스침'이다 — 3연타**
```
누가 스치고 / 미안하단 말은 머릿속에만 / 나도 스치듯 안 했고
```
몸이 스치고, 말도 스치고, 나도 스친다.
**이 도시에서 관계는 전부 스침뿐**이라는 걸 한 단어의 반복으로 못 박는다.

**③ 갈 수 있는데 안 간다 — 곡 최대의 여백**
```
한 블록 건너면 지하철인데
나는 신호를 기다리며
```
집에 갈 수 있다. 한 블록이면 된다. 그런데 신호를 기다리며 서 있다.
**왜 안 가는지 끝까지 말하지 않는다.** 집에 가기 싫은 건지, 여기 있고 싶은 건지 자기도 모른다.
> `신호`는 「네온」의 `초록불은 오지 않고`와 이어지고, A05 「신호대기」로 연결됨

**④ 마지막에 주어를 지운다**
```
저기 있었던 적이 있나
```
`나는`이 빠지며 누구 얘긴지 흐려진다. **자기가 저 무리에 속했던 적이 있는지조차 확신 못 하는 상태.**
시점도 지하철 입구(밖)에서 번화가(안)를 돌아보는 것으로 이동 — 이미 떠나온 자리를 본다.

### 보이스 스펙 적용 (1.6절)
| 항목 | 적용 |
|---|---|
| ① 몸으로 말하기 | `귀가 아픈 것만 같아` `어깨가 자꾸 누구랑 부딪히고` `가방끈만 꼭 쥐고` |
| ② 초구체적 명사 | 간판·신발·전단지·전광판·가방끈·지하철 입구 |
| ③ 자조 | `비켜 주는 건 어렵지 않지` — 웃음기 없이 **이미 익숙해진 담담함** |
| ④ 자기를 밖에서 | `저기 있었던 적이 있나` (주어 생략) |
| ⑤ 끊어치는 리듬 | `귀가 아픈 것만 같아`(8) `가방끈만 꼭 쥐고`(7) |
| ⑥ 질문형 마무리 | `여기서 뭘 보고 있나` / `저기 있었던 적이 있나` |
| ⑦ 설명 없음 | 감정어 0건 |

> 💡 `것만 같아`는 흐리는 게 아니라 **아픈지 아닌지도 확신 못 하는 감각 둔화**로 읽힌다.
> 💡 `어렵지 않지`는 자기 비하(`잘하네`)가 아니라 **오래 해서 쉬워진 일**이라는 체념. 무감각 톤에 더 맞음.

> ⚠️ A03의 화장실 거울과 **같은 장치(반사면으로 자기 보기)를 쓰지 않았다.**
> 반사 대신 **행동**(비켜주기)으로 처리해 중복을 피함.

### 개작 기록
- `줄 선 사람들은 갈 데가 있고` → **`신발이 모두 다른데`** (시선 높이 정보가 생김)
- `발밑은 아까부터 축축하고` → **`한 블록 건너면 지하철인데`** (안 가는 이유의 여백)
- `나는 저기 있었던 적이 있나` → **주어 생략**
- 18음절 최장행 `…머릿속에만 스쳐` → **`…머릿속에만`**으로 트림 (부르기 문제만 해소, 3연타 리듬은 유지)
