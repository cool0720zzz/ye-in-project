# A05 「신호대기」 — 가사 + Suno 세팅

- **세트**: A (도시의 밤 · 1인칭)
- **감정**: **답답함·억압** — 규율에 갇힘. 어기고 싶지만 못 함
- **장면**: 건널목. 다들 다른 신발로 다른 곳을 향하는데 **신호라는 병목 앞에 다 같이 멈춰 있다**
- **메커니즘**: **차가 한 대도 안 오는데 빨간불이라 아무도 안 건넌다.** 누가 먼저 가면 다 갈 텐데 아무도 안 간다
- **BPM/무드**: 100, 마이너 시티팝. 정체된 긴장감. 중간에 **정적 구간**
- **언어**: 한국어

> **연결**: A04 아웃트로 `나는 신호를 기다리며`에서 바로 이어짐. `신발` 모티프도 A04에서 계승
> **3부작 시작**: A05 억눌림 → A06 터짐 → A07 결과

## 설계 노트

### ⭐ 구조 — 정적 구간(Break)
```
A01 네온     Intro - V1 - Pre - Chorus - V2 - Pre - Chorus - Bridge - Chorus - Outro
A02 워커홀릭  Intro - V1 - Chorus - V2 - Chorus - Bridge - [Synth Solo] - Chorus - Outro
A03 회식     Intro - V1 - Chorus - V2 - Chorus - Bridge - Outro   (끝 후렴 없음)
A04 번화가   Intro - Chorus - V1 - Chorus - V2 - Bridge - Chorus - Outro   (후렴 선행)
A05 신호대기  Intro - V1 - Chorus - V2 - [Break] - Bridge - Chorus - Outro   ← 정적 삽입
```
**기다림을 음악으로 실연한다.** V2 뒤에 소리가 거의 멎는 구간을 두어,
듣는 사람도 같이 신호를 기다리게 만든다.

### 중심 장치 — 차가 없는데 아무도 안 간다
```
차는 한 대도 안 오는데
빨간불은 안 바뀌고
누가 먼저 가면 다 갈 텐데
아무도 안 가네
```
**규율을 어겨도 아무 일 안 생기는 걸 다들 아는데, 아무도 먼저 안 간다.**
억압을 설명하지 않고 **아무도 움직이지 않는 사실**만 놓는다.

### 몸이 먼저 반응한다 (스펙 ①·④)
```
발이 반 걸음 나갔다 들어오고
...
그걸 아는데도 발은 여기 있고
```
**충동과 억제가 전부 발에서 일어난다.** 내가 참는 게 아니라 발이 알아서 돌아온다.
`고개를 좌우로 두 번` — 차 오나 확인하는 행동. 건너고 싶다는 걸 말 안 하고 보여줌.

### 보이스 스펙 적용 (1.6절)
| 항목 | 적용 |
|---|---|
| ① 몸으로 | `발이 반 걸음 나갔다 들어오고` `고개를 좌우로 두 번` `발은 여기 있고` |
| ② 초구체 명사 | 신발·빨간불·폰 화면·숫자(카운트다운)·초록불 |
| ③ 자조 유머 | **없음 (의도적 포기)** — 아래 트레이드오프 참조 |
| ④ 자기를 밖에서 | 발을 3인칭처럼 다룸 |
| ⑤ 끊어치는 리듬 | **`아무도 안 가네`**(6) **`초록불`**(3) |
| ⑥ 질문형 마무리 | 질문 대신 **미완결 어미** `향하는 곳이 다른데` |
| ⑦ 설명 없음 | 감정어 0건 (단, `보는 눈이 많거든`은 의도적 예외) |

### ⚠️ 트레이드오프 기록 — 자조 대신 감시
```
어길 수가 없어. 보는 눈이 많거든
```
초안은 `준법정신 하나는 확실하네`(자조 유머)였으나 **의도적으로 교체.**
- **얻은 것**: 못 어기는 이유가 **사회적 감시**임을 명시. 주제(억압)가 선명해짐
- **잃은 것**: 곡의 유일한 웃음 포인트. 스펙 ③이 비고, `~거든`이라는 설명조 어미가 들어옴
- → 곡으로 들어본 뒤 무겁기만 하면 `준법정신 하나는 확실하네`로 복구 가능

### 아웃트로 — 같이 기다렸지만
```
초록불
다들 각자 갈 데로 흩어지고
결국 기다린 건 같이지만
향하는 곳이 다른데
```
`초록불` 한 단어로 병목이 풀린다. 그리고 남는 건 **군중 속 연대의 허상** —
**같이 기다렸다고 같은 편은 아니다.** `~는데`로 끝나 닫지 않는다.

### A04와의 연결
- A04 아웃트로 `나는 신호를 기다리며` → A05가 그 장면
- `신발` 모티프 계승: A04 `줄 선 사람들 신발이 다 다른데` → A05 `신발은 앞 신호처럼 다 다르고`
  > 💡 두 곡 다 **`다 다르(른)`**으로 어미가 맞물려, 세트를 이어 들을 때 모티프가 **소리로도** 연결됨 (14음절 트림의 부수 효과)
- ⭐ **`앞 신호처럼`** = **이미 한 번 신호를 놓치고 또 기다리는 중.**
  한 번 기다리는 것과 두 번째 기다리는 건 답답함의 크기가 다르다
