# B01 「주말에」 — 가사 + Suno 세팅

- **세트**: B (가족·집) — A세트(도시의 밤)와 다른 축
- **감정**: **상주처럼 굴다가 들통나는 곡.** 절절하게 듣다가 마지막에 **피식**
- **⭐ 화자의 정체**: **그냥 멀리 사는 건데 사별한 사람처럼 구는 애같은 마음.**
  속이는 게 목적이 아니라 **화자가 스스로 오버하고 있는 게** 웃긴 것
- **장면**: 혼자 사는 방. 청양고추 팍팍 넣은 **엄마 된장찌개**가 먹고 싶다
- **⭐ 설계 원칙**: **모든 줄이 사별처럼 들리지만 전부 "혼자 산다"는 사실.** 거짓말이 하나도 없다
- **금지어**: `보고 싶다` `그립다` `사랑` — **한 번도 쓰지 않는다** (쓰는 순간 "그냥 가면 되잖아"가 되어 미끼가 죽는다)
- **BPM/무드**: 90, 따뜻한 어쿠스틱 시티팝. 장례식처럼 무겁게 쌓다가 마지막에 툭
- **언어**: 한국어

> 💡 **출처**: 학생 때 처음 술 마시고 엄마한테 전화해서 "청양고추 팍팍 넣은 된장찌개" 먹고 싶다고 울었던 실제 기억
> ⚠️ **속이면 안 된다.** 장례·영정·유품처럼 *죽음을 지어내는 문장*을 쓰는 순간 반전이 사기가 된다

## 설계 노트

### ⭐⭐ 이 곡의 전부 — 상주처럼 굴되, 거짓말은 안 한다

청자는 **엄마가 돌아가신 줄 안다.** 그런데 **사실이 아닌 문장이 하나도 없다.**

| 줄 | 사별처럼 들림 | 실제 뜻 |
|---|---|---|
| `그 칼칼한 게 없더라` | 다시는 못 먹음 | 밖에선 그 맛이 안 남 |
| **`엄마 없이 혼자가 처음이라`** | **사별 후 첫 시기** | **혼자 산 게 처음** |
| **`엄마 없이, 나 혼자`** | **남겨짐** | **자취 중** |
| `아무도 안 알려줬잖아` | 상실을 감당하는 법 | 자취하는 법 |
| **`엄마 없이 맞는 첫 겨울이`** | **첫 기일 전 겨울** | **혼자 나는 첫 겨울** |
| `밥이 잘 안 넘어가고 / 밤에는 자꾸 깨더라` | 애도 증상 | 자취 스트레스 |
| `그런 것도 다 옛날이야` | 고인의 추억 | 몇 년 전 일 |

> ⚠️ **절대 금지**: 장례·영정·유품·`다시는 못 봐` 같은 **죽음을 지어내는 문장.**
> 그 순간 반전이 **사기**가 되고 청자가 배신감을 느낀다.
> 평범한 사실만 놓고 **청자가 알아서 오해하게** 둬야 반전이 **귀엽게** 넘어간다.

### ⭐ `잖아` — 애같은 마음의 증거
```
아무도 안 알려줬잖아
```
`~잖아`는 **어리광 어미**다. 다 큰 사람이 혼자 사는 걸 두고
"아무도 안 알려줬잖아"라고 투정하는 순간, **화자가 애처럼 굴고 있다는 게 새어나온다.**
반전을 알고 다시 들으면 **이 줄이 제일 웃기다.**

### ⭐ 아웃트로 — `전화를 걸었어`가 미끼를 끊는다
```
전화를 걸었어
별일은 아니고
된장찌개나 먹으러
주말에 내려간다니까

그래 밥 해둘게, 하고
엄마가 먼저 끊네
```
- ⭐ **`전화를 걸었어`** — **여기서 미끼가 끊긴다.** 3분 내내 사별한 사람처럼 굴다가
  **그냥 전화를 건다.** 죽은 사람한테는 전화가 안 걸린다는 걸 청자가 그제야 깨달음
- `별일은 아니고` — **아무도 안 물었는데 먼저 방어.** 여기서부터 들킴
- `된장찌개나` — `~나`가 **하찮게 만드는 조사.** 별거 아닌 척
- `내려간다니까` — **전언체**라 남 얘기처럼. 자기 결정인데 자기도 인정 안 함
- ⭐⭐ `그래 밥 해둘게` — 딸은 *된장찌개*라 했는데 엄마는 ***밥***을 해둔다.
  **엄마가 다 알아들었다는 증거.** 아무도 그 말을 하지 않는데 둘 다 안다
- ⭐⭐ **`엄마가 먼저 끊네`** — 엄마도 **별일 아닌 척한다.** 길게 안 끌고 툭 끊는다.
  (끊자마자 장 보러 갈 사람이다.) **모녀가 똑같이 무심한 척하는 게 마지막 농담이자 가장 아픈 지점**

### 청자 반응 설계
```
V1 ~ 마지막 후렴   무겁게 ───────────↗   (엄마가 돌아가신 줄 안다)
`전화를 걸었어`     엇? ↘                (…어? 전화가 되네?)
`된장찌개나 먹으러`  피식                 (아 그냥 멀리 사는 거였구나 ㅋㅋ)
`엄마가 먼저 끊네`  뭉클                 (둘 다 아닌 척하는 게 보임)
```
**웃기기만 하면 실패.** 마지막 행에서 **다시 뭉클해져야** 성공.

### 라임 설계
| 장치 | 적용 |
|---|---|
| **후렴 모음 통일** | 처음이**라** / 알려줬잖**아** / 알아 **가** / 나 혼**자** — 전부 **`ㅏ` 계열** 착지 |
| **후렴 수미상관** | 1행 `**엄마 없이** 혼자가 처음이라` ↔ 4행 `**엄마 없이**, 나 혼자` — 같은 말로 열고 닫아 훅이 됨 |
| **`-더라` 모티프** | V1 `그 칼칼한 게 없**더라**` / V2 `자꾸 깨**더라**` |
| **`엄마 없이` 3연타** | 후렴 1행·4행 + V2 3행(`엄마 없이 맞는 첫 겨울`) — 곡을 관통하는 미끼 문구 |

> 💡 `-더라`는 **직접 겪고 알게 된 걸 전하는 어미**라 이 프로젝트의 건조한 관찰자 목소리와 맞는다.
> 한국어에서 영어식 각운을 억지로 맞추면 동요처럼 유치해진다 — **어미 통일 + 모음 계열 통일**이 실제 라임 장치.

### ⭐ 운율 설계 (실측값)
```
             음절
V1       9 / 11 /  9 /  8
V2       8 /  8 / 10 /  8
후렴    11 /  9 /  9 /  7      ← 3회 완전 동일, 끝행 7음절로 떨어뜨려 여백 확보
브릿지   7 /  7 /  9 /  9 / 9
아웃트로 6 / 6 / 8 / 9 // 8 / 7
```
- **후렴 3회 완전 동일** → Suno가 같은 멜로디 재사용 → 훅이 생긴다
- **V1/V2 음절 차이 1/3/1/0** → 2행만 3음절 벌어짐 (`어딜 가 봐도 그 맛이 아니야` 11 ↔ `밤에는 자꾸 깨더라` 8)
- **전체 최장 11음절**
- **아웃트로가 6→7음절로 내려앉음** → 힘이 빠지며 끝나는 게 수치로도 보임

### ⭐⭐ 여백 원칙 — 「엄마가 딸에게」에서 배운 것
양희은·김창기 「엄마가 딸에게」가 우는 방식은 **벨팅이 아니라 여백**이다.
```
공부해라(4) / 아냐 그건 너무 교과서야(11)
성실해라(4) / 나도 그러지 못했잖아(10)
너의 삶을 살아라(8)
```
**짧은 구절 + 긴 구절이 교대**하고, 핵심은 전부 **8음절 이하**다.
말이 빽빽하면 **감정이 들어갈 자리가 없다.**

> ⚠️ **절절하게 만들려면 음절을 늘리는 게 아니라 줄여야 한다.**
> 실제로 초안에서 후렴을 11/11/13/11로 늘렸더니 오히려 감정이 죽었고,
> **11/9/9/7**로 줄이자 마지막 `엄마 없이, 나 혼자`(7)에 늘여 부를 공간이 생겼다.

> ⚠️ **음절은 손으로 세지 말고 반드시 측정할 것.** 이전 초안에서 손으로 센 값이 실측과 최대 2음절 어긋났다.
> 한국어는 음절만으로 리듬이 안 잡힌다 — **어절(띄어쓰기 단위) 수**를 함께 볼 것.

### 구조 — A세트와 다른 골격
```
A01 네온   Intro - V1 - Pre - C - V2 - Pre - C - Bridge - C - Outro
A03 회식   Intro - V1 - C - V2 - C - Bridge - Outro        (끝 후렴 없음)
B01 주말에 Intro - V1 - C - V2 - C - Bridge(5행) - C - Outro(6행)
```
- **프리코러스 없음**: 애도는 뜸 들이지 않는다. 바로 후렴으로
- **브릿지 → 후렴 → 아웃트로**: 브릿지(추억)로 최고조 → 후렴 한 번 더 → 아웃트로가 힘을 뺌 → **낙차 최대**

### 잔소리 소재는 뺐다
엄마의 불안/잔소리(`위험한 건 하지 말라던`)는 **삭제.**
주선율이 **몸의 기억**(된장찌개·첫 술·첫 겨울)인데 잔소리를 넣으면
**원망이 섞여 애도 톤이 깨진다.**
> 엄마의 불안 소재는 **B세트 별도 곡**으로 분리 (후보)
