# B01 「주말에」 — 가사 + Suno 세팅

- **세트**: B (가족·집) — A세트(도시의 밤)와 다른 축
- **감정**: **상실** — 단, 죽음이 아니라 **떨어져 사는 상실**. 그래서 더 말 못 함
- **장면**: 혼자 사는 방. 엄마의 불안이 싫었는데, 지금은 아무도 나를 불안해하지 않는다
- **⭐ 엔진**: **내가 싫어했던 게 바로 지금 없어서 힘든 것.** 엄마의 불안 = 감옥인 줄 알았는데 안전망이었다
- **금지어**: `보고 싶다` `그립다` `사랑` — 한 번도 쓰지 않는다
- **BPM/무드**: 90, 따뜻한 어쿠스틱 시티팝. A세트의 네온 밤과 달리 **집·살결의 온도**
- **언어**: 한국어

> ⚠️ **후회·반성으로 가지 말 것.** "말 들을걸"은 이 곡이 아니다. 그냥 **없다는 사실**만.

## 설계 노트

### ⭐ 중심 장치 — 같은 후렴, 뒤집힌 의미
후렴은 **한 글자도 안 바뀌는데** 브릿지를 지나면 **정반대로 들린다.**

| 위치 | 어떻게 들리나 |
|---|---|
| 1·2번째 후렴 | 지긋지긋한 **잔소리 목록** (싫었던 것) |
| **3번째 후렴** (브릿지 뒤) | 같은 말이 **나를 붙잡아주던 손**으로 들림 |

가사를 고쳐서 감정을 뒤집는 게 아니라, **브릿지가 맥락을 바꿔 같은 말의 의미를 뒤집는다.**
「네온」의 극성 뒤집기가 *단어*를 뒤집었다면, 이 곡은 *맥락*을 뒤집는다.

### ⭐ `오르지도 내리지도 않게` — 이중의미
- 표면: **등락 없는 안전한 삶**을 살라는 엄마의 바람
- 이면: **심전도의 평평한 선.** 안전한 삶 = 죽은 것 같은 삶
- 브릿지에서 다시 쓰이며 **세 번째 뜻**이 열림 — `떨어져도 아무도 모른다`
  → 오르내림이 없길 바란 건 **떨어질 때 받아주려던 것**이었다

### 구조 — A세트와 다른 골격
```
A01 네온     Intro - V1 - Pre - C - V2 - Pre - C - Bridge - C - Outro
A03 회식     Intro - V1 - C - V2 - C - Bridge - Outro        (끝 후렴 없음)
B01 주말에   Intro - V1 - C - V2 - C - Bridge - C - Outro    ← 프리코러스 없음, 짧은 반전 아웃트로
```
- **프리코러스 제거**: 잔소리는 뜸 들이지 않는다. 바로 후렴으로 쏟아져야 함
- **아웃트로 3행(6/7/9음절)**: 앞이 다 무거우므로 **짧고 싱겁게** 끝내 김을 뺀다

### ⭐ 아웃트로 — 변명이 다 들킨다
```
별일은 아니고
주말에 내려간대
반찬이나 좀 받아 오게
```
- `별일은 아니고` — **먼저 방어부터 한다.** 아무도 안 물었는데
- `내려간대` — **`~대`(전언체)로 남 얘기처럼.** 자기 결정인데 자기도 인정 안 함
- ⭐ `반찬이나 좀 받아 오게` — **핑계.** 보고 싶어서가 아니라 반찬 때문이라고
  → 세 줄이 전부 **부인(否認)**인데, 그래서 다 들킨다. `보고 싶다`를 한 번도 안 쓰고 도착

### 몸·구체
| 항목 | 적용 |
|---|---|
| 초구체 명사 | 부재중 다섯 통 / 뉴스 사고 / 약 / 반찬 |
| 몸의 감각 | `열이 나면 혼자 약을 사 와` — 아플 때 혼자인 것 |
| 자조 | `반찬이나 좀 받아 오게` (곡의 유일한 웃음, 근데 웃다가 목이 멤) |
| 질문형 | `무슨 뜻이었는지` (답하지 않음) |
| 감정어 | **0건** — `싫다` `그립다` `보고 싶다` 전부 미사용 |

### 음절 설계 (이 곡부터 규율 적용)
```
V1   9 / 10 / 10 /  9
V2  10 / 11 / 10 /  9     → 행별 차이 1/1/0/0
후렴 9 /  8 /  9 / 10     (3회 모두 동일)
```
**V1·V2를 ±1음절로 맞춰** Suno가 같은 멜로디를 재사용할 수 있게 설계.
후렴 최장 10음절로 **부담 없는 범위**.

### 딸/아들 차이 메모
엄마의 불안은 **핵심(안정 지향·도전 말라)은 같고 표현 지점이 다르다.**
- 아들 → 직업 안정성·성공
- 딸 → 거기에 **안전 걱정이 더 얹힘** (밤길·혼자 사는 것·건강·연락)

예인이 여성 화자이므로 **디테일은 연락·건강·혼자 사는 것**으로 잡되,
핵심(`오르지도 내리지도 않게`)은 그대로 둔다.
