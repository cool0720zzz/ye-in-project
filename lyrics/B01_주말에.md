# B01 「주말에」 — 가사 + Suno 세팅

- **세트**: B (가족·집) — A세트(도시의 밤)와 다른 축
- **감정**: **그리움 → 반전.** 절절하게 듣다가 마지막에 **피식** 웃게 되는 곡
- **장면**: 혼자 사는 방. 엄마 김치·따뜻한 밥·팔짱 끼고 걷던 길이 다 그립다
- **⭐ 설계 원칙**: **모든 줄이 죽음처럼 들리지만, 사실은 전부 "멀리 산다"는 얘기다. 거짓말이 하나도 없다**
- **금지어**: `보고 싶다` `그립다` `사랑` — 한 번도 쓰지 않는다
- **BPM/무드**: 90, 따뜻한 어쿠스틱 시티팝. 절절하게 쌓다가 마지막에 힘을 뺀다
- **언어**: 한국어

> ⚠️ **후회로 가지 말 것.** "말 들을걸"은 이 곡이 아니다.
> ⚠️ **속이면 안 된다.** 죽었다고 *말하는* 순간 반전이 사기가 된다. 청자가 알아서 오해하게 둘 것

## 설계 노트

### ⭐⭐ 이 곡의 전부 — 오해하게 두되, 거짓말은 안 한다

청자는 **엄마가 돌아가신 줄 안다.** 그런데 가사에 **사실이 아닌 문장이 하나도 없다.**

| 줄 | 죽음처럼 들림 | 실제 뜻 |
|---|---|---|
| `엄마 김치가 떨어졌어` | 다시는 못 얻음 | 그냥 다 먹음 |
| `이제 잘 기억이 안 나` | 세월이 흐름 | 오래 못 먹어서 |
| `아직 거기 있을 텐데` | 나만 못 감 | 고향에 있음 |
| **`엄마 없이 뭐 하고 있나`** | **부재 = 죽음** | 옆에 없다는 뜻 |
| `너무 오래된 것 같아서` | 유품 사진 | 그냥 옛날 사진 |

> ⚠️ **절대 금지**: 죽음을 *암시하는 문장을 지어내는 것*(장례·유품·못 만남 등).
> 그 순간 반전이 **사기**가 되고 청자가 배신감을 느낀다.
> 우리는 **평범한 문장을 놓고 청자가 알아서 오해하게** 둔다. 그래야 반전이 **귀엽게** 넘어간다.

### ⭐ 아웃트로 — 엄마가 알아듣는다
```
별일은 아니고
김치나 좀 받으러
주말에 내려간다니까
엄마가 밥 해놓는대
```
- `별일은 아니고` — **아무도 안 물었는데 먼저 방어.** 여기서부터 들킴
- `김치나 좀` — `~나`가 **하찮게 만드는 조사.** 별거 아닌 척
- `내려간다니까` — **전언체**로 남 얘기처럼. 자기 결정인데 자기도 인정 안 함
- ⭐⭐ **`엄마가 밥 해놓는대`** — 딸은 *김치*라고 했는데 엄마는 ***밥***을 해놓는다.
  **엄마가 다 알아들었다는 증거.** 아무도 그 말을 하지 않는데 둘 다 안다
  > 💡 **곡에서 엄마 목소리(행동)가 처음 등장하는 순간.** 앞은 전부 화자의 독백이었다가,
  > 마지막에 엄마가 나타나면서 **"살아 있다"가 확정**되고 동시에 **가장 따뜻해진다**

### 청자 반응 설계
```
V1~Bridge   절절 ────────────────↗  (엄마가 돌아가신 줄 안다)
Outro 1~3행 툭 ↘  (어? 살아계시네)
Outro 4행   피식 + 뭉클  (엄마가 다 알고 밥을 해놓는다)
```
**울다가 웃게 만드는 게 목표.** 반전이 웃기기만 하면 실패,
`밥 해놓는대`에서 **다시 뭉클해져야** 성공.

### 잔소리는 V2에만 — 그리움의 목록 중 하나로
```
위험한 건 하지 말라던
그 잔소리가 없으니까
겁이 나도 말할 데가 없어
```
엄마의 **불안**이 싫었는데, 없으니까 **겁날 때 말할 데가 없다.**
싫었던 그물이 사실 **안전망**이었다는 걸 설명하지 않고 배치로 보여줌.
> ⚠️ 이건 **곡의 주선율이 아니라 한 겹**이다. 주선율은 김치·밥·팔짱 같은 **몸의 기억**

### 구조 — A세트와 다른 골격
```
A01 네온   Intro - V1 - Pre - C - V2 - Pre - C - Bridge - C - Outro
A03 회식   Intro - V1 - C - V2 - C - Bridge - Outro        (끝 후렴 없음)
B01 주말에 Intro - V1 - C - V2 - C - Bridge - C - Outro    ← 프리코러스 없음
```
- **프리코러스 없음**: 그리움은 뜸 들이지 않는다. 바로 후렴으로
- **브릿지 → 후렴 → 아웃트로**: 브릿지(사진첩)로 감정을 최고조로 올리고,
  후렴을 한 번 더 때린 **직후에** 아웃트로가 힘을 뺀다 → **낙차 최대화**

### 음절 설계
```
V1    9 / 9 /  9 / 8
V2    9 / 9 / 10 / 8     → 행별 차이 0/0/1/0
후렴  8 / 8 / 10 / 9     (3회 모두 동일)
브릿지 8 / 6 /  7 / 9    (짧게 떨어뜨려 아웃트로 준비)
아웃트로 6 / 7 / 9 / 8
```
전체 **최장 10음절**. A세트 곡들보다 짧고 담백하게 — 어쿠스틱 편성에 맞춤.
