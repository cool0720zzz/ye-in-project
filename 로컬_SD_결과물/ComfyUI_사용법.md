# ComfyUI로 예인 이미지 뽑기 — 사용법

## 1. 켜기
프로젝트 폴더의 **`예인_ComfyUI_실행.bat`** 더블클릭.
- 검은 창이 뜨고 잠시 후 브라우저가 자동으로 열림 (`localhost:8188`).
- 처음엔 30초~1분 걸릴 수 있음. 끄려면 검은 창을 닫으면 됨.

## 2. 예인 워크플로우 불러오기 (처음 한 번만)
ComfyUI 화면에서:
1. 화면에 파일을 **드래그**하거나, 좌측/상단 메뉴의 **Load(불러오기)** 클릭
2. **`로컬_SD_결과물/예인_워크플로우_ComfyUI.json`** 선택
3. 노드 그래프가 뜸 — 체크포인트(Juggernaut), Lightning LoRA, 프롬프트, 설정이 다 세팅돼 있음

## 3. 생성
- 우측/하단의 **Queue Prompt(큐)** 버튼 클릭 → 약 40~50초 후 이미지 나옴
- 결과는 `ComfyUI/output/yein/` 폴더에 자동 저장

## 4. 바꿔가며 뽑기
- **프롬프트**: 초록색 큰 텍스트 박스(위=긍정, 아래=부정) 수정
- **다른 얼굴/포즈로**: KSampler 노드의 **seed**가 "randomize"라 누를 때마다 자동으로 바뀜
- **더 고화질**: KSampler의 `steps`를 8 → 그대로 두고, 최고 화질 원하면 아래 "고화질 모드" 참고

## 핵심 설정 (이미 박혀있음 — 참고용)
| 항목 | 값 | 이유 |
|---|---|---|
| 체크포인트 | juggernautXL_v9 | 실사 특화 |
| LoRA | sdxl_lightning_8step | 8스텝으로 단축 |
| steps | 8 | Lightning 전용 |
| cfg | 1.0 | Lightning은 낮은 cfg |
| sampler | euler | Lightning 권장 |
| scheduler | sgm_uniform | Lightning 권장 |

⚠️ **Lightning은 cfg를 1.0으로 둬야 함.** 7 같은 값 넣으면 이미지가 타버림.

## 한글 프롬프트로 뽑기 (번역 노드)
`ComfyUI_Custom_Nodes_AlekPet` 설치됨 → 한글 타이핑하면 자동 영어 번역.

1. **`예인_한글프롬프트_ComfyUI.json`** 불러오기
2. 위쪽 프롬프트 박스에 **한글로** 입력 (예: "카페에 앉아 창밖을 보는 예인, 크림 니트")
3. **실행(Queue)** → 번역 후 생성 (~40초)

⚠️ **인터넷 필요** (구글 무료 번역 사용). 오프라인이면 안 됨.
⚠️ **"울프컷"은 옷에 늑대가 새어나올 수 있음** → 한글로도 "레이어드 단발"로 쓰는 게 안전.
💡 번역 품질이 애매하면, 그 부분만 영어로 섞어 써도 됨 (한/영 혼용 가능).
부정 프롬프트(아래 박스)는 영어 그대로 두세요 — 이미 세팅돼 있음.

## 그록 이미지 수정하기 (img2img)
그록으로 만든 이미지를 **실사로 승격**하거나 화풍을 바꾸려면:

1. **`예인_img2img_ComfyUI.json`** 불러오기
2. **`이미지 로드`(Load Image)** 노드에서:
   - 노드의 **업로드 버튼** 클릭 → 그록 이미지 선택
   - (또는 그록 파일을 캔버스에 드래그하면 이 노드가 자동 생성됨)
3. **KSampler의 `노이즈 제거량`(denoise)**으로 수정 강도 조절:
   - `0.2~0.3` → 살짝 다듬기 (그록 그림 거의 유지)
   - **`0.5`** → 구도·포즈 유지 + 화풍을 실사로 승격 (기본값, 권장)
   - `0.7+` → 구도만 참고, 거의 새로 그림
4. **실행(Queue)** → `ComfyUI/output/yein_img2img/`에 저장

⚠️ **입력 이미지 해상도가 곧 출력 해상도.** 그록 이미지가 너무 크면(예: 2048) 느려지고
   VRAM을 많이 씀. 1024 근처로 리사이즈하면 무난.
💡 denoise가 낮을수록 실제 스텝이 줄어 **풀 생성보다 빠름** (0.5면 약 25~50초).

## 고화질 모드 (선택)
최고 화질이 필요하면 LoRA를 빼고(LoraLoader 노드 우클릭 → Bypass), KSampler를
`steps 25 / cfg 6 / scheduler karras`로. 대신 43초 → 약 213초(3.5분).

## 프롬프트 팁 (늑대 버그 관련)
`wolf-cut`, `puppy` 같은 단어는 **옷에 동물 그림으로 새어나감.**
→ 헤어는 `shaggy layered bob`, 얼굴은 `large round charming eyes`로 표현.
부정 프롬프트에 `animal on sweater, graphic print`가 이미 들어가 있음.
