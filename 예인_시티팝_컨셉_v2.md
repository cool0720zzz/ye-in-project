## 4. 세트리스트 리테마 (한국어 시티팝)

기존 "예인의 이야기 / 청자에게 전하는 이야기 / 인스트루멘탈" 3분류 구조는 유지하되, 테마를 도시·밤·노스탤지어로 재편.

### A세트 — 도시의 밤 (1인칭) *(2026-07-20 전면 재설계)*

> **설계 원칙 3가지**
> ① **밤의 도시에만 있는 것** — 낮에도 있거나 도시 밖에도 있으면 탈락 (카세트·서리·우산 등 폐기)
> ② **메커니즘이 있을 것** — 장면 나열(`자정의 드라이브`)이 아니라, 그 대상이 **실제로 하는 일**이 감정과 겹쳐야 함
> ③ **감정을 흩을 것** — 전부 이별 정서로 깔면 단조로움. 소진·소외·동경·연대·안도까지 분산

| # | 제목 | 감정 | 장면 → 메커니즘 |
|---|------|------|------|
| A01 | **네온** | 이별의 점멸 | 깜빡인다 → 마음도 오락가락 ✅완성 |
| A02 | **워커홀릭** | **소진·공허** | 다 꺼진 사무실에 모니터 불빛만 내 얼굴을 비춘다 → 열심히 하는데 어디로 가는지 모르겠음 |
| A03 | **회식** | **가면·소외** | 웃는 표정을 유지하다 화장실 거울 앞에서 풀린다 → 여기 있는데 여기 없는 느낌 |
| A04 | **번화가** | **동경·막막함** | 저 불빛은 전부 누군가의 것이고 나는 지나가는 사람 → 이 도시에 내 자리가 있나 |
| A05 | **신호대기** | **답답함·억압** | 다들 다른 신발로 다른 곳을 향하는데 **신호라는 병목 앞에 다 같이 멈춘다.** 규율을 어기고 싶지만 못 함 |
| A06 | **막차** | **해방·일탈** ☀️ | 억압에 짓눌린 나를 푸는 시간. **막차를 타려다 돌아서서 더 놀기로 한다** |
| A07 | **첫차** | **공허** | 그렇게 더 놀았는데 **남은 게 없다** |

> 🕐 **시간 순으로 하룻밤이 완성됨**: 야근 → 회식 → 번화가 → 건널목 → 막차 → 첫차
> (네온은 첫 곡이자 전체 테마 선언)

> ⭐ **A05~A07은 인과가 있는 3부작**
> ```
> A05 억눌림  건널목에서 막힘, 규율을 못 어김
> A06 터짐    막차를 포기하고 돌아감 (유일한 업템포·밝은 곡)
> A07 결과    놀았는데 남은 게 없음
> ```
> A04 아웃트로 `나는 신호를 기다리며`가 A05로 바로 이어지고, `신발` 모티프도 A04→A05로 연결됨.
> **A06이 앨범의 유일한 업템포** — 전곡 미드템포 마이너라 여기서 확실한 대비를 만든다.

### B세트 — 너에게 (2인칭) *(2026-07-20 전면 재설계)*

> **A와의 구분**: A는 **내 얘기**(내가 겪는 밤), B는 **너에게 건네는 말**. 시점으로 명확히 갈림.
> **위로를 말로 하지 않는다** — 전부 사물·행동이 대신함. 같이 먹기, 같이 앉아있기, 켜져 있는 표시등, 멀어지는 택시를 보기.
> 기존 제목(`오늘도 수고했어` `빛나는 내일` `혼자가 아니야`)은 **전부 블랙리스트급 상투구라 폐기.**

| # | 제목 | 감정 | 장면 → 메커니즘 |
|---|------|------|------|
| B01 | **야식** | **말 없는 동석** | 늦은 밤 마주 앉아 말없이 먹는다 → 무슨 일인지 안 물어볼게 |
| B02 | **옥상** | **침묵의 편안함** | 나란히 앉아 야경만 본다 → 아무 말 안 해도 되는 시간 |
| B03 | **비상구** | **조용한 확신** | **모든 불이 꺼져도 그것만은 켜져 있다** → 다 어두워도 나갈 길은 있어 |
| B04 | **택시** | **배웅** | 태워 보내고 뒷유리가 멀어질 때까지 본다 → 끝까지 지켜보는 마음 |
| B05 | **24시** | **무조건적 기다림** | 아무도 안 와도 밤새 열려 있다 → 언제 와도 돼 |
| B06 | **소등** | **하루를 닫아줌** | 불을 하나씩 끈다 → 이제 자도 돼 (엔딩) |

> ⚠️ 감정을 분산할 것 — 동석/침묵/확신/배웅/기다림/재움. **전부 "힘내"류면 6곡이 똑같아짐**

## 5. 비주얼 / 영상 무드 (개정)

### 5.1 톤앤매너
- **아트 스타일**: **실사(포토리얼) 시네마틱** — 아나모픽, 얕은 심도, 네온 블룸·헤일레이션, 35mm 필름그레인, 틸-마젠타 그레이딩
  → ⚠️ 실사는 루프 관용도가 낮아 **루프 규칙(수동적 인물·아웃포커싱 배경) 준수가 필수**
- **컬러**: 마젠타·시안 네온, 퍼플/딥블루 야경, 선셋 그라데이션 (페르소나별 팔레트는 캐스트 문서 참조)
- **질감**: 필름 그레인, 라이트 리크, VHS·시티팝 앨범 커버 감성
- **배경**: 도시 야경, 하이웨이, 옥상, 네온 간판, 자판기 골목
- **타이포**: 80s 레트로 산세리프 / 일본 시티팝 앨범풍 로고

### 5.2 영상 방식 — 루프 영상 (풀 MV 폐기)

풀 뮤직비디오(곡당 12~19클립)를 폐기하고, **짧은 심리스 루프 영상을 곡 길이만큼 반복**하는 방식으로 전환.
시티팝/로파이 유튜브 씬의 표준 포맷이며 제작량이 ~1/10로 감소.

- **생성 도구**: Grok (클립 최대 15초)
- **원칙**: 큰 동작 X, 앰비언트 모션 O (네온 깜빡임, 창밖 비, 지나가는 헤드라이트, 머리카락·김 흔들림, 예인이 살짝 리듬 타기)
- **무페이드 원칙(중요)**: 크로스페이드는 루프 주기마다 미세 깜빡임=피로감을 만든다. **후처리로 이음새를 숨기지 않고, 애초에 루프되는 영상을 뽑는다.**
  - ① 시네마그래프화(카메라 고정, 앰비언트만 움직임) ② 자연 루프 프롬프트("마지막=첫 프레임") ③ 방향성은 균일 반복 환경(터널/연속 광류)으로 대체
  - 상세 프롬프트/방식: `video/그록_루프_프롬프트.md`
- **심리스 처리(무페이드)**:
  - 방법 A) 부메랑 — 정방향+역방향 이어붙여 시작/끝 자동 일치 (대칭 모션용)
  - 방법 B) 하드 루프 — 자연 루프 클립을 그대로 반복 (`build`, 후처리 없음)
  - 크로스페이드는 최후의 폴백으로만
- **곡 길이만큼 확장 + 오디오 합성 (FFmpeg)**:
  ```bash
  ffmpeg -stream_loop -1 -i loop15s.mp4 -i song.mp3 \
         -map 0:v -map 1:a -shortest -c:v libx264 -c:a aac out.mp4
  ```
