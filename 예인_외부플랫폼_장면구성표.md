# 예인 외부 플랫폼 장면 구성표

> **Kling AI / Runway / Hailuo / Pika 등에서 직접 생성할 때 사용하는 레퍼런스**
> 각 장면별 사용할 이미지, 프롬프트, 설정값, 저장 파일명을 정리했습니다.
> 프롬프트를 그대로 복사해서 해당 플랫폼에 붙여넣기 하면 됩니다.

---

## 공통 설정

| 항목 | 값 |
|------|-----|
| 해상도 | 16:9 (1920×1080 또는 1280×720) |
| 길이 | 5초 기본 (인트로만 10초 가능) |
| 모션 강도 | 중간~낮음 (자연스러운 움직임) |
| 카메라 | 고정 (카메라 움직임 없음) |

**공통 스타일 키워드 (모든 프롬프트 끝에 추가):**
```
warm studio lighting, soft natural tones, shallow depth of field,
cinematic look, plants in background, wooden floor,
white backdrop with large windows, gentle sunlight
```

## 생성 후 품질 체크 리스트

각 클립 생성 후 확인할 사항:

- [ ] 예인의 얼굴이 원본과 일관성 있는가? (너무 다른 얼굴이 되지 않았는가)
- [ ] 움직임이 자연스러운가? (관절이 꺾이거나 손이 이상하지 않은가)
- [ ] 의상이 일관적인가? (올블랙 유지)
- [ ] 배경이 원본 스튜디오와 비슷한가?
- [ ] 영상 길이가 적절한가? (너무 짧거나 길지 않은가)
- [ ] 해상도가 최소 720p 이상인가?
- [ ] 다른 클립들과 이어붙였을 때 자연스러운가?

**마음에 들지 않으면:**
- 같은 프롬프트로 3~5회 재생성 (매번 결과가 다름)
- 프롬프트에 "subtle movement", "very gentle" 추가 (움직임 과할 때)
- 프롬프트에 "maintaining exact appearance" 추가 (얼굴 변형 시)
