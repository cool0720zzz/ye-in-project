## 3. Suno AI 프롬프트 (공통 베이스)

```
Korean female vocal singing in english, slight korean accent, non-native pronunciation,
clear syllables, slightly accented english, young breathy female whisper, quiet vocals
buried behind instruments, lazy mumbled articulation, youthful airy tone, slightly slurred
dreamy delivery, Soft cafe jazz, acoustic guitar, piano, brushed drums, upright bass,
```

## 4. 콘텐츠 구성 및 비율

### 4.1 곡 카테고리

| 카테고리 | 설명 | 비율 목표 |
|----------|------|-----------|
| **A. 예인의 이야기** | 생활 속 따뜻함을 노래 (아이의 눈빛, 아빠 손의 온기, 포근한 날씨 등) | 40% |
| **B. 청자에게 전하는 이야기** | 위로와 응원 (고생했어, 너를 응원해, 내일 더 빛날 거야 등) | 40% |
| **C. 가사 없는 음악** | 인스트루멘탈 (쉬는 시간) | 20% |

### 4.2 실제 편성 구조 (2시간 기준)

**기본 블록 패턴:**
- 보컬 7곡 (~3분/곡 = 21분) → 인스트루멘탈 2곡 (~3분/곡 = 6분) = **27분/블록**

**전체 배치:**

| 순서 | 구간 | 곡 수 | 시간 | 카테고리 |
|------|------|-------|------|----------|
| 1 | 예인의 이야기 #1 | 7곡 | ~21분 | A |
| 2 | 쉬는 시간 #1 | 2곡 | ~6분 | C |
| 3 | 청자에게 전하는 이야기 #1 | 7곡 | ~21분 | B |
| 4 | 쉬는 시간 #2 | 2곡 | ~6분 | C |
| 5 | 예인의 이야기 #2 | 8곡 | ~24분 | A |
| 6 | 쉬는 시간 #3 | 2곡 | ~6분 | C |
| 7 | 청자에게 전하는 이야기 #2 | 8곡 | ~24분 | B |
| 8 | 쉬는 시간 #4 | 2곡 | ~6분 | C |
| **합계** | | **38곡** | **~114분** | |

> **인트로/아웃트로/전환 영상 포함 시 총 ~120분 이상**

**실제 비율 계산:**
- A (예인의 이야기): 15곡 × 3분 = 45분 → **39.5%**
- B (청자의 이야기): 15곡 × 3분 = 45분 → **39.5%**
- C (인스트루멘탈): 8곡 × 3분 = 24분 → **21%**
- **달성 비율: 약 4 : 4 : 2 ✓**

## 5. 영상 구성 상세

### 5.1 인트로 시퀀스 (6~10초)

영상의 맨 처음 1회 재생:

| 시간 | 장면 | 소스 |
|------|------|------|
| 0-2초 | 예인이 의자에 앉는 모습 | AI 영상 생성 (IMG_2868 기반) |
| 2-4초 | 측면 3/4 각도에서 마이크 위치 세팅하는 모습 | AI 영상 생성 (IMG_2868 기반) |
| 4-6초 | "안녕하세요, 예인입니다" 발랄하게 인사 | AI 영상 생성 (IMG_2869 기반) |
| 6-8초 | 허밍 시작하며 눈을 감는 모습 | AI 영상 생성 → 정지 프레임 전환 |

### 5.2 곡 전환 패턴 (보컬곡)

각 보컬곡마다 반복되는 구조:

```
[허밍 시작 + 눈 감는 영상 2-3초]
    ↓
[노래 시작 3-5초 후 → 노래하는 사진으로 정지 (곡 끝까지)]
    ↓
[다음 곡 허밍 시작 + 눈 감는 영상 2-3초]
    ↓
  (반복)
```

### 5.3 쉬는 시간 전환 패턴 (인스트루멘탈 구간)

```
[마지막 보컬곡 종료]
    ↓
["잠깐 쉬어갈게요" 멘트 + 500ml 물병 들고 화면 밖으로 나가는 영상 3-4초]
    ↓
[인스트루멘탈 곡 재생 (카페 배경 정지 이미지)]
    ↓
[인스트루멘탈 마지막 곡 끝나기 전 → 다시 의자에 와서 앉는 영상 3-4초]
    ↓
[허밍 시작 + 눈 감기 → 다음 보컬곡 시작]
```

## 11. 곡별 가사 작성 가이드라인

### 공통 규칙
1. 영어 가사이되 한국인이 발음하기 자연스러운 단어 선택
2. 복잡한 자음 클러스터 피하기 (e.g., "strengths" → "strong" 선호)
3. 짧고 반복적인 후렴구로 중독성 확보
4. 가사 밀도 낮게 (여백 많이, 허밍/라라라 활용)
5. 곡당 가사 분량: verse 2개 + chorus 2회 + bridge 1개 정도

### A세트 (예인의 이야기) 가사 톤
- 1인칭 시점 ("I remember...", "I can feel...")
- 감각적 묘사 중심 (냄새, 온도, 빛, 촉감)
- 과거 회상 + 현재 감사의 교차

### B세트 (청자에게 전하는 이야기) 가사 톤
- 2인칭 시점 ("You did so well...", "I see you...")
- 따뜻한 위로 어조
- 명령형 지양, 제안형 선호 ("You can rest" > "Rest now")
