## 3. 루프 영상 (Kaggle LTX) — 진행 중

- **케글 무료 T4가 루마보다 낫다(사용자 확정)**. 절차 문서: `무료GPU_루프영상_노트북.md`
- 노트북: kaggle.com/code/**linoailab/notebook058aeab969** (GPU T4×2·인터넷 ON 세팅 저장됨)
  - 셀1 = 자체완결 파이프라인: pip 설치 → LTX-Video 로드 → **base64 내장 앵커**(워커홀릭 키보드 스틸 512×320) → seed 1234 생성 → PSNR 루프컷 → `/kaggle/working/loop.mp4`
  - 검증된 시드 스캔 결과: 1234=31.95dB, 777=32.17, 2026=30.77, **4321=32.61 (BEST)** — 전부 "명확히 튐" 등급이라 **부메랑 폴백 필수**
  - 부메랑은 PC에서: `ffmpeg` 정방향+역방향 concat (loop_best.mp4 기반)
- **캐글→PC 파일 회수 (원격 검증된 방법)**: 다운로드 버튼·Output 패널·외부 업로드 전부 막힘 →
  **콘솔 base64 청크 릴레이**: `print('C1S'+B64[0:45000]+'C1E')` → `get_page_text`로 회수 → Write로 파트 저장 → PC에서 조립+MD5 대조.
  loop_best.mp4=167KB=b64 222,708자=**45K×5청크**. 콘솔 Clear 후 청크당 1회.
- ⚠️ **함정**: ① Run/Ctrl+Enter는 세션 부팅 중이거나 빈 셀 선택 시 조용히 무시됨 — 실행 확인은 "Cancel Run" 버튼 등장으로 ② 노트북 뷰가 빈 화면이면 새로고침 ③ **세션 유휴 리셋되면 /kaggle/working 파일 소실** — 완성 즉시 회수 ④ **GPU 주 30시간 쿼터 — 작업 끝나면 세션 전원 OFF 필수**

## 4. 갤러리 파이프라인

```
PYTHONIOENCODING=utf-8 python scripts/gallery_publish.py \
  --file <경로> --type image|loop|video|lyricvideo --id <고유id> --title "..." --verdict "..." [--final]
git add gallery/manifest.json && git commit && git pull --rebase && git push
```
- 같은 `--id` 재발행 = 카드 교체(리워크 반영에 사용)
- 썸네일은 manifest에 data URI 내장 → 소스 파일은 커밋 불필요(`grok_stills/`는 gitignore)
- 폰 결정은 `gallery/decisions*.json` 커밋으로 돌아옴 → **작업 전 git pull로 확인** (choice: continue/rework/hold + notes)
- push 전 `git pull --rebase` 습관화 (폰 결정 커밋과 자주 충돌)

## 5. Grok Imagine 스틸 규칙

- 프롬프트 = 꼬리표C(예인 인물, `video/그록_루프_프롬프트.md`) + 씬 + 꼬리표A(실사 영상미), **16:9·이미지 모드·품질**
- 루프 앵커용은 "인물 정지+환경만 움직일 여지"로 구도 잡기, `no readable text`
- 전송 성공 판정 = **탭 타이틀이 프롬프트로 바뀜**. 실패 시 입력창 재클릭→타이핑→↑버튼
- 다운로드는 **상세 페이지 완전 로드 후** 우하단 아이콘. 파일명 `grok-image-<id>.jpg` 또는 `<id>.jpg`
- 발행된 스틸 원본: `grok_stills/` (A02 키보드, A03 배시시, A04 인파, A05 빨간불, B01 전화)
