## 1. 곡 생성 워크플로우

### 일일 작업 흐름
1. **곡 생성 (매일 ~20곡)**
   - Suno Create 페이지에서 곡별 가사 + 스타일 입력
   - 곡_생성_가이드.md 참고하여 곡별 스타일 키워드 적용
   - 각 곡당 2개 버전 자동 생성됨
   - Ye_in 페르소나 + 블렌딩 스타일 고정

2. **리뷰 & 선별**
   - 생성된 곡 들어보기
   - 마음에 드는 버전에 Suno에서 **따봉(좋아요)** 표시
   - 안 좋은 버전은 무시 또는 재생성

3. **따봉 곡 필터링 & 다운로드**
   - Suno Library에서 좋아요 필터링
   - 선별된 곡만 다운로드
   - 곡별 폴더로 정리

4. **후속 작업 (선별된 곡만)**
   - 커버 이미지 생성 (예인 캐릭터 포함)
   - 영상화 작업 (프리픽 활용)

- **페르소나**: Ye_in persona
- **기본 스타일 베이스**:
