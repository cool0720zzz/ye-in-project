# 예인 프로젝트 — 작사 가이드

> **목적**: 38곡 전체가 같은 밀도의 시적 언어를 갖도록. 상투적 가사 방지.
> 마지막 수정: 2026-07-20

---

## 0. 왜 이 문서가 생겼나

「네온」 초고가 **장면을 설명만 하고 변형하지 않아** 상투적이었음.
"네온 불빛 아래 나 홀로 걷는 밤" / "네 목소리가 들려와" / "오래 기억할게"
→ 전부 이미 들어본 문장이라 **걸리는 데(울림)가 없음.**

---

## 1. 레퍼런스와 기법

| 레퍼런스 | 배울 것 |
|---|---|
| **한로로** | 국문학과 출신, "가사를 시처럼" 씀. **함축·은유**. 시집의 울림을 가사에 옮김 |
| **황인찬** | **쉬운 단어 + 구어체 단순 구문**으로 풍부한 의미 공간. 쉽게 읽히는데 의미는 안 단순함. **여백** |
| **기형도** | 도시의 밤·단절. **감정을 말하지 않고 사물이 대신** 지게 함 (예: '외투' 이미지로 내면 표현) |

---

## 1.5 ⭐ 이 프로젝트의 목소리 (모든 원칙의 상위)

> **느끼는 바를 직접 말하지 않는다.**
> **건조하게, 남의 일인 것처럼 말하는데 — 사실은 아픈 것.**

이게 「네온」에서 나온 톤이고, 앞으로 모든 곡이 지켜야 할 기준이다.

**검증법**: 가사에서 **감정어를 전부 지웠을 때도 아픈가?**
「네온」에는 `그립다/외롭다/아프다`가 **하나도 없다.** 전부 사실·관찰·질문인데 아프다.

| 톤을 무너뜨리는 것 | 톤을 지키는 것 |
|---|---|
| `바쁘면 괜찮은 줄 알았는데` | `점심은 자리에서 먹었고 / 저녁은 먹었는지 모르겠어` |
| `그냥 아무 느낌이 없어` | `끝나는 칸이 원래 없었어` |
| `그거면 오늘은 됐어` | `알람은 여섯 시에 맞춰뒀어` |

왼쪽은 **감정을 설명**하고, 오른쪽은 **사실만 놓는데 감정이 새어나온다.**
설명하는 순간 듣는 사람이 할 일이 없어진다.

## 1.6 ⭐ 예인 보이스 스펙 (기본값)

1.5절이 **원칙**이라면, 이건 실제 곡 3편(「네온」「워커홀릭」「회식」)에서 추출한 **구체적 지문**이다.
초안은 **처음부터 이 결로** 뽑을 것.

> ⚠️ **기본값이지 절대 규칙이 아님.** 소재에 따라 조절될 수 있다.
> (예: B세트 2인칭 위로곡은 자조 유머 비중이 낮아질 수 있음)

### ① 감정이 아니라 몸으로 말한다
생리 상태만 말하고 감정은 듣는 사람이 채우게 둔다.
- `이상하네 내가 흔들리는지 / 사람들이 흔들리는 건지`
- `초점이 하나도 안 맞네` · `눈꺼풀이 이렇게 무거웠나` · `맞장구는 몸이 먼저 치고`

### ② 한국 생활의 초구체적 명사
추상명사 대신 **손에 잡히는 물건**. 즉시 그림이 그려진다.
- `숙취해소제` `머리끈` `캔맥주` `2차` `분노의 타이핑` `알람`

### ③ 자조적 유머를 반드시 하나 심는다
무거움만으로 가지 않는다. **웃기면서 아픈** 지점을 만든다.
- `내 키보드만 분노의 타이핑 중` · `문을 열지. 직장인. 간다.`

### ④ 자기를 밖에서 본다
'나'를 주어에서 밀어내고 **3인칭이나 신체 부위**로 바꾼다. 자기 소외가 시점으로 구현된다.
- `거울 앞에 텅 빈 채 웃는 / 직장생활하는 어른이 있네`
- `시야가 그 이름을 지우는 밤` · `시선은 자꾸 없는 쪽을 봐`

### ⑤ 끊어치는 리듬
긴 문장만 쌓지 않고 **짧은 행을 브레이크로** 던진다.
- `꿀꺽`(2음절) · `그 와중에도`(5) · `문을 열지. 직장인. 간다.`

### ⑥ 질문형으로 닫는다
단정을 피하고 물음으로 여백을 남긴다.
- `무거웠나` `맞춰뒀던가` `나여야 하는 건가` `흔들리는 건지`

### ⑦ 설명은 쓰지 않는다 (가장 강한 금지)
아래 유형은 **예외 없이 전부 폐기됐다.**
- ❌ `바쁘면 괜찮은 줄 알았는데` `그냥 아무 느낌이 없어` `그거면 오늘은 됐어` `이거 하나 마시려고`

---

### 한 줄 요약
> **건조한 관찰 + 생리 상태 + 초구체적 한국 명사 + 자조적 유머 + 끊어치는 리듬 + 질문형 마무리**

### 초안 작성 시 자가진단
- [ ] 감정어 대신 **몸의 상태**로 썼는가
- [ ] **손에 잡히는 명사**가 충분한가 (추상명사만 있으면 실패)
- [ ] **웃음 포인트가 하나** 있는가
- [ ] '나'를 밖에서 보는 시점이 한 번은 나오는가
- [ ] **짧은 행**이 리듬을 끊어주는가
- [ ] 마무리가 **질문형**인가
- [ ] **설명하는 문장이 하나도 없는가**

> 💡 문어체(시적 은유) 아니라 **구어체(생활 디테일 + 자조)**. 이게 두 언어의 갈림길이다.

## 2. 3대 원칙

### ① 감정어를 직접 쓰지 말고 사물이 지게 한다
- ❌ 그립다 / 외롭다 / 슬퍼 / 보고 싶어 / 기억할게
- ✅ 그 감정을 **떠안은 사물**을 놓는다
  - 그리움 → `쏟은 사람은 여기 없는데` (번진 색이 대신 짐)
  - 소멸감 → `간판의 글자가 하나씩 꺼져 / 누가 내 이름을 지우는 밤`

> ⚠️ **앵커는 "감정 진술"이 아니라 "질문 또는 사실"이다 (실패에서 배움)**
> 해석이 어렵다고 **감정어를 넣으면 톤이 무너진다.** 「네온」에는 감정어가 **하나도 없다** —
> `그립다/외롭다/아프다` 전무하고 전부 사실·관찰·질문인데도 아프다.
> - ✅ 앵커로 쓸 것: **질문**(`나는 어디쯤부터 꺼져있었나`) / **관찰**(`시선은 자꾸 없는 쪽을 봐`)
> - ❌ 앵커로 쓰면 안 되는 것: `바쁘면 괜찮은 줄 알았는데` `그냥 아무 느낌이 없어` `그거면 오늘은 됐어`
>
> **"나 무감각해"라고 말해버리면 무감각이 아니다.**

### ② 장면을 설명하지 말고 변형한다
- ❌ 네온이 예쁘다, 거리가 물들었다 (본 대로 적기)
- ✅ 네온을 **다른 것으로 재정의**: `빌린 빛이라 오래 가지 못해`
- ✅ 관계를 **뒤집기**: `도시는 밤마다 다시 켜지는데 / 나는 어디쯤에서 꺼진 걸까`

### ②-b ⭐주체를 '나'에서 감각기관·신체로 옮긴다
슬픔의 주체를 사람이 아니라 **눈·시선·시야·손** 등으로 바꾸면, 감정이 **통제 밖에서 일어나는 일**이 되어 훨씬 깊어진다.

| 평범 | 전환 | 효과 |
|---|---|---|
| 누가 내 이름을 지우는 밤 | **시야가** 그 이름을 지우는 밤 | 지우는 게 외부가 아니라 **내 안**에서 일어남 |
| **나는** 자꾸 없는 쪽을 봐 | **시선은** 자꾸 없는 쪽을 봐 | 내가 보는 게 아니라 **시선이 제멋대로** 감 = 통제 불능 |

> 기형도가 즐겨 쓰는 방식. "내가 슬프다"가 아니라 **몸의 일부가 알아서 반응**하게 둔다.

### ②-c 모티프를 하나로 꿰맨다
이미지가 흩어지면 밀도가 안 생긴다. **한 계열의 이미지가 곡 전체를 관통**해야 척추가 생김.
- 「네온」의 축: **흐림 → 번짐 → 흘러내림** (시야가 뭉개짐 → 색이 번짐 → 빛을 흘려냄)
- 아웃트로가 그 실을 마무리하도록 설계할 것

### ②-d 논리 명제 금지 — 사고의 질감으로
규칙을 세우고 그 안에서 추론하면 **듣는 순간 계산을 요구**한다. 노래는 되감기가 안 된다.
- ❌ 규칙 세우기 + 추론 (읽으면 정교한데 들으면 안 따라옴)
- ✅ **실제로 드는 생각의 결** — 새벽에 드는 생각은 논증하지 않는다. **의아해하고, 되묻고, 맴돈다**

### ②-e ⭐감정은 한 상태로 고정되지 않는다
이별·그리움을 쓸 때 **한 상태만 그리면 거짓말이 된다.** 실제 마음은 오락가락한다 —
멀쩡한 밤도 있고, 내가 더 빛나는 날도 있고, 생각조차 안 나다가 갑자기 훅 온다.

**해법: 극성 뒤집기 대구** — 같은 문장을 곡의 다른 지점에서 **한 단어만 뒤집어** 반복
```
V2      불이 켜진 쪽은 나여야 하는 건가
Bridge  불이 꺼진 쪽이 나여야 하는 건가
```
같은 사람이 두 번 다르게 묻는다 = 흔들림이 **구조 자체로** 드러남.
진폭을 *설명*하지 않고 *실연*하며, 후렴 밖에 **두 번째 훅**까지 만든다.

### ③ 쉬운 말로 쓰되 다 말하지 않는다
- 어려운 한자어·문어체 금지. **입에 붙는 구어체**로.
- 단, **결론을 닫지 말 것.** 여백이 울림을 만든다.
  - `나는 조금 늦게 켜져` ← 회복을 직접 말하지 않음

## 3. 상투구 블랙리스트 (쓰지 말 것)

| 금지 | 이유 |
|---|---|
| 네온 불빛 아래 / 별빛 아래 / 달빛 아래 | 무한 반복된 구문 |
| 너의 목소리가 들려와 | 클리셰 |
| 오래 기억할게 / 잊지 않을게 | 감정 직진술 |
| 눈물이 흐른다 / 가슴이 아파 | 신체 반응 상투 |
| 시간이 흘러도 / 세월이 지나도 | 시간 관용구 |
| 분홍빛 파란빛 물든 | 색 나열만으로 정서 대체 |
| 반짝이다 흐려져 | 추상적 상태 서술 |
| 우리의 추억 / 그날의 우리 | 지시어만 있고 이미지 없음 |
| 하지만 어쩔 수 없지 | 체념으로 감정을 닫아버려 여백이 사라짐 |

### 문법·표기 주의
- **`~함은` `~됨은` 등 명사형 문어체 금지** — 노래로 안 붙고 뒷 구절과 문법이 끊김
  - ❌ `머물다 떠나감은 / 빌려온 빛으로 오래 가지 못해`
  - ✅ `머물다 가 / 빌려온 빛이라 오래 가지 못해`
- **영어는 훅 한 구절에만.** 여러 줄에 흩뿌리면 촌스러워짐 (같은 훅의 반복은 무방 — 각인 효과)

## 4. 이미지 발상 뱅크 (도시의 밤)

사물이 감정을 지게 하는 소재. **그대로 쓰지 말고 변형해서 쓸 것.**

| 사물 | 지게 할 수 있는 것 |
|---|---|
| 꺼지는 간판 글자 | 이름·존재의 소멸 |
| 젖은 아스팔트에 번진 색 | 쏟은 사람의 부재 |
| 신호등 초록불 | 오지 않는 허락 / 대답 |
| 자판기 | 밤에 유일하게 깨어 있는 것 / 넣어도 안 나오는 것 |
| 캔의 온도 | 식어가는 관계 |
| 막차·정류장 | 놓친 것 / 기다림의 자리 |
| 김 서린 유리에 쓴 글씨 | 번지는 것 = 흘러가는 것 |
| 가로등마다 생기는 그림자 | 여러 개의 나 |
| 편의점 불빛 | 밤에 유일하게 열린 문 |
| 발목·손끝·목덜미 | 계절·기억이 남아 있는 부위 |

> **신체 부위에 감각을 남기는 기법**이 특히 유효: `여름은 발목에만 아직 뜨거워`

## 5.5 음절·운율 (Suno 기준)

### 결론: 엄격한 음절 맞추기는 불필요
전통적 작사에서 음절을 세는 건 **이미 만들어진 멜로디에 얹기 때문**. 작사가도 작곡가의 음표를 못 바꿈(리듬이 틀어지면 그루브가 무너짐).
**Suno는 반대 — 가사에서 멜로디가 나온다.** 맞춰야 할 고정 멜로디가 없으므로 제약이 뒤집힘.

> 추세도 같은 방향: K-pop은 **"한 음표당 한 음절"이라는 제한적 작사법에서 이미 벗어났고**, 한영 혼용이 늘며 음절 수보다 **강세·리듬감** 중심으로 이동.

### 그래도 지켜야 할 3가지

| # | 원칙 | 이유 |
|---|---|---|
| ① | **반복 구간은 구조를 동일하게** | 후렴이 3회 반복되면 음절 구조가 같아야 멜로디가 깨끗이 반복됨 |
| ② | **대응 구간(V1↔V2)은 음절 수 근사** | 같은 멜로디가 두 벌스에 얹히므로 |
| ③ | **극단적 편차 회피** | 긴 줄 옆 짧은 줄 → Suno가 욱여넣거나 늘임. **낙차 5음절 이상 주의** |

### 실측 예시 — 「네온」
```
[Verse 1]  12 / 11 / 10 / 10        [Verse 2]  11 / 11 / 12 / 11   ✅ 근사
[Pre-C]     8 / 10  (2회 동일)                                      ✅
[Chorus]   12 / 13 / 12 / 12 / 5   (3회 동일)                       ✅ 균일
[Bridge]    9 / 12 / 12 / 12                                        ✅
[Outro]    13 / 15 / 8 / 11                                         ⚠️ 15→8 낙차
```
**아웃트로 보정 예** (검증된 실측치, 의미 유지):
| 원본 | 대안 | 비고 |
|---|---|---|
| `함께 꺼지고 켜지는 네온 빛에 기대어` **15** | `함께 꺼지고 켜지는 네온에 기대어` **14** | `빛` 삭제(앞줄 "빛은"과 중복 회피), 네온 유지 |
| `아직 흐린 시야 속에` **8** | `아직도 흐린 시야 속에서` **10** | 낙차 완화 |
→ **낙차 7 → 4**로 축소

### 음절 세는 법
⚠️ 셸 인자로 한글을 넘기면 **인코딩이 깨져 숫자가 틀림.** 반드시 **UTF-8 파일로 읽을 것.**
```python
# lines.txt 에 한 줄씩 저장 후
import re, io
for ln in io.open('lines.txt', encoding='utf-8'):
    print(len(re.findall('[가-힣]', ln)), ln.strip())
```

### 발음 편의성 (음절 수보다 중요할 때가 있음)
- 받침 없는 열린 모음이 **길게 끌기 쉬움** → 후렴 마지막 음절에 유리
- 자음 충돌(ㄱ+ㅋ, ㅅ+ㅆ 등) 연속은 **부르기 어려움**
- 반드시 **입으로 불러볼 것** — 눈으로 읽어서는 안 걸림

## 6.5 개작 태도 (실패 기록 — 반복 금지)

「네온」 작업에서 실제로 겪은 실패:
작성자가 자기 가사에 **의문을 표한 것**("해석이 붙어야 이해되지 않나?")을 **폐기 요청으로 오독**해,
3라운드에 걸쳐 전면 대안을 쏟아냈고 **전부 기각**됨.

**그래서 지킬 것**
1. **원안을 먼저 방어해볼 것** — 원안이 가진 질감이 곧 정서일 수 있다.
   「네온」의 경우 *빙빙 도는 논리*가 곧 *맴도는 마음*이었다.
2. **대안은 한 번에 소수만.** 여러 개를 쏟아내면 전부 "같은 명제의 다른 포장"이 되기 쉽다.
3. **빗나가면 더 뽑지 말고 먼저 물을 것** — 무엇이 어긋났는지(밋밋한가/설명적인가/정서가 다른가).
4. **정서 축을 함부로 재정의하지 말 것.** 작성자가 처음에 말한 축이 대개 맞다.
   (「네온」은 최초 제시된 *점멸*이 정답이었는데, 중간에 *비대칭*·*그리움 증가*로 두 번 잘못 잡았다가 복귀)

> 최종 해법은 결국 **작성자가 원문의 한 단어를 고쳐서** 나왔다.
